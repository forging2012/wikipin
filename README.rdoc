== README

This is a hypermedia API for wikipedia geolocated articles.

Once a GET request is sent to the API entry point. IP is geo-located and a path between wikipedia articles is created.

Still W-I-P.

DB Dumps available at:

https://www.dropbox.com/sh/pz9qrb33lx1uibr/AACe9WkYWC8iP1zbw89J4hoaa?dl=0
